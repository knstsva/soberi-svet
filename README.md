# СОБЕРИ СВЕТ 

Мини-игра на Python с использованием библиотеки Pygame.  
Цель игры — собирать светящиеся точки быстрее соперника и набрать необходимое количество очков за ограниченное время.

---

## Описание игры

Игрок управляет персонажем и собирает светящиеся точки на карте.  
Доступны два режима:

- 1 игрок (игра на время)
- 2 игрока (соревнование на одной клавиатуре)

Также доступна настройка сложности (время раунда):

- Лёгкий — 120 секунд  
- Средний — 60 секунд  
- Сложный — 30 секунд  

Побеждает тот, кто набирает больше очков или достигает цели быстрее.

---

## Управление

Игрок 1:
- W — вверх
- S — вниз
- A — влево
- D — вправо

Игрок 2:
- Стрелка вверх — вверх
- Стрелка вниз — вниз
- Стрелка влево — влево
- Стрелка вправо — вправо

---

## Запуск проекта

Установить зависимости:

pip install pygame

Запустить игру:

python game.py

---

## Используемые технологии

- Python 3.x
- Pygame
- Random (генерация объектов)
- Math (расчёты расстояний и движения)
- OS (работа с файлами звука)

---

## Звук

Игра поддерживает:
- фоновую музыку (music.mp3)
- звук клика (click.wav)
- звук сбора точки (hit.wav)

Если аудиофайлы отсутствуют, игра работает без звука.
